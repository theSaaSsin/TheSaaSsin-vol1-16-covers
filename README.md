## What's in this repo

This repo hosts the **cover-art renderer** and, over time, the openly-shareable build artefacts. The full prompt packs, architecture diagrams, and system walkthroughs ship as premium downloads on Gumroad.

```
TheSaaSsin-vol1-16-systems-/
├── generate.js     # Cover art renderer — primary layout
├── generate2.js    # Cover art renderer — alternate composition
└── README.md
```

### Render the covers

```bash
node generate.js
# outputs vol_01.png … vol_16.png into ./output/
```

Puppeteer auto-installs on first run.
