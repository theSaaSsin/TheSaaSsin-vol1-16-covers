# TheSaaSsin Vol. 1 — 16 Systems

A **16-volume system-architecture library** for SaaS founders, operators, and service businesses. Each volume is a full build: the architecture diagram, the prompt pack that generates it, the playbook for running it, and the cover art for the release.

> Not a course. Not a template dump. A buildable spec + the prompts to produce the output end-to-end.

## What each volume ships

Every one of the 16 volumes contains:

- **System architecture** — the diagram + component map for the workflow
- **Prompt pack** — the exact prompts (and sequencing) to generate or run the system end-to-end
- **Build guide** — the walkthrough for implementation, stack-agnostic where possible
- **Cover art** — release art for the volume (generated by the scripts in this repo)

## Cover art — how to render

The two generate scripts use Puppeteer to produce the 16 release covers from the volume list.

```bash
node generate.js     # primary layout -> output/vol_01.png … vol_16.png
# or
node generate2.js    # alternate composition
```

Puppeteer auto-installs on first run. Output lands in `./output/`.
